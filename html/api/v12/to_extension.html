POST /api/1.2/to_extensions

Creates a Traffic Ops extension.

Authentication Required: Yes

Role(s) Required: None

Request Parameters

Parameter

Type

Description

name

string

version

string

info_url

string

script_file

string

isactive

string

additional_config_json

string

description

string

servercheck_short_name

string

type

string

Request Example

{
      "name": "ILO_PING",
      "version": "1.0.0",
      "info_url": "http://foo.com/bar.html",
      "script_file": "ping",
      "isactive": "1",
      "additional_config_json": "{ "path": "/api/1.2/servers.json", "match": { "type": "EDGE"}",
      "description": null,
      "servercheck_short_name": "ILO"
      "type": "CHECK_EXTENSION_BOOL",
}

Response Properties

Parameter

Type

Description

alerts

array

A collection of alert messages.

>level

string

Success, info, warning or error.

>text

string

Alert message.

Response Example

{
 "alerts": [
    {
       "level": "success",
       "text": "Check Extension loaded."
    }
 ],
}
